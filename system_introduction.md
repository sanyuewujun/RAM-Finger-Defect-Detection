# 基于深度学习的内存条金手指不良判断系统
### 毕业设计答辩汇报

**汇报人**：李毅
**专业**：智能科学与技术 B2212
**学院**：信息工程学院

---

## 1. 项目背景与行业痛点

> 内存条（RAM）金手指质量直接影响产品可靠性及后续装配效果。

- 📍 **现状**：传统人工目视检验效率低、准确率波动大。
- 📍 **痛点**：高强度质检环境下人员易疲劳，无法满足大批量生产需求。
- 📍 **对策**：基于深度学习构建自动化、可视化的不良品识别系统。

---

## 2. 系统目标与核心价值

- [x] **全自动化判定**：替代人工，实现金手指合格性自动归类。
- [x] **生产线适配**：支持文件夹持续扫描与批量推理。
- [x] **工程化落地**：提供 GUI 交互界面，降低操作门槛。
- [x] **模型迭代**：内置完整的训练与测试评估闭环。

---

## 3. 技术架构设计

| 模块层级 | 核心组件 | 技术实现说明 |
| :--- | :--- | :--- |
| **数据处理层** | `dataset.py` | 兼容复杂路径，支持图像增强与归一化 |
| **算法核心层** | `models.py` | 适配 ResNeXt 与 Swin Transformer 网络 |
| **训练调度层** | `trainer.py` | AdamW 优化器 + OneCycleLR 动态学习率 |
| **界面交互层** | `inference_ui.py` | 基于 PyQt5 的可视化操作与日志回显 |

---

## 4. 深度学习模型选型

🛠️ **兼顾精度与效率的双重保障**

- **卷积神经网络 (CNN)**:
  - `ResNeXt50/101_32X4D`：利用分组卷积提升特征提取深度。
- **视觉 Transformer (ViT)**:
  - `Swin_B / Swin_V2_B`：通过移动窗口机制捕捉复杂纹理特征。

> 优势：支持根据实际算力资源灵活切换模型，满足不同工业场景需求。

---

## 5. 训练策略与优化机制

- **损失函数**：采用 `CrossEntropyLoss` 处理分类任务。
- **优化策略**：`AdamW` 权重衰减结合 `OneCycleLR` 学习率调度，加速收敛。
- **防御性设计**：
  - **早停机制 (Early Stopping)**：防止模型在有限样本下过拟合。
  - **数据增强**：通过水平/垂直翻转、缩放增强模型泛化性。

---

## 6. GUI 设计与功能演示

📍 **面向生产现场的可视化管理**

1. **实时推理**：支持单张调试与万级别图片的批量分拣。
2. **持续扫描**：自动监控待检目录，实现“随采随检”。
3. **分拣自动化**：判定结果自动移动至 `AIResult/OK` 或 `NG`。
4. **日志回显**：在 GUI 中实时绘制 `Loss` 与 `Accuracy` 曲线。

---

## 7. 项目难点与创新点

- 💡 **鲁棒性增强**：解决了工业路径中复杂字符导致的读取失败问题。
- 💡 **体验创新**：打破算法“黑盒”，将深度学习全生命周期日志在 UI 端可视化。
- 💡 **闭环设计**：集成了从数据采集、增强、训练到推理分拣的全流程功能。

---

## 8. 总结与未来扩展

🚀 **项目总结**：系统实现了工业场景下内存条金手指不良品的自动判定，具备高可用性与算法先进性。

- [ ] **深度方向**：引入划痕、氧化、缺口等细分缺陷识别。
- [ ] **性能方向**：部署至边缘计算设备，结合工业相机硬件接口。
- [ ] **智能方向**：引入自动超参搜索，进一步提升识别精度。
